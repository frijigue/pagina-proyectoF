Línea de tiempo de creación de página web

Primera versión

Se propuso un primer diseño, donde se cumplía con la usabilidad del sistema. Al informarnos sobre páginas web personales se observo que debemos dar y promocionar nuestra redes sociales, los cuales son lugares para la comunicación

Segunda versión

Presentamos la propuesta final. Nuestra propuesta tiene una paleta de colores simple: negro, blanco y amarillo el cual es un color con el que se identifica el autor. La propuesta tiene una esctructura básica: header para mostrar el logo del autor, nav para mostrar las pestañas de referencias que nos apoyan en la navegación del sitio, podemos observar que en cada pestaña se activa con diferente color la página en la que nos encontramos. Aside, esta parte decidio agregarlo el autor, para el tema de promocionar en este cso sus redes sociales y un footer para presentar quién realizo la página
